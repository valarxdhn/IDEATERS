## 🚀 How to Get Started (3 Simple Steps)

### Step 1: Read QUICK_START.md (15 min)
```
1. System requirements check
2. Clone & setup
3. Install dependencies
4. Run application
5. Login & test
```

### Step 2: Run Locally (15 min)
```bash
# Backend (Terminal 1)
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python edulrew_backend.py

# Frontend (Terminal 2)
cd frontend
npm install
npm start

# Open browser: http://localhost:3000
```

### Step 3: Test With Sample Data (10 min)
```
1. Create 3 test students
2. Add academic data
3. Add socioeconomic data
4. View risk analysis
5. Check recommended schemes
6. Record intervention
```
